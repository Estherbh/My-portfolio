# Donor Journey & Membership Funnel Analysis

## Overview

This portfolio project demonstrates how donor acquisition, first-gift conversion, repeat giving, and membership uptake can be analyzed in one stakeholder-friendly view.

The scenario uses fictional HopeTrack campaign data to mirror the kind of reporting logic expected in fundraising, communications, and CRM-linked analytics roles.

## What this project shows

- donor journey analysis across the funnel
- one-time vs monthly donor conversion logic
- cross-channel donor acquisition comparison
- retention and upgrade analysis
- clear recommendations for non-technical stakeholders

## Files

- Report page: [index.html](index.html)
- Sample dataset: [donor-journey-data.csv](donor-journey-data.csv)

## Funnel covered

The report follows this simplified journey:

1. Session acquired
2. Donation page visit
3. Donation started
4. First donation completed
5. Second donation within 90 days
6. Monthly membership activated

## Key questions answered

- Which channels bring the most qualified donor traffic?
- Where is the biggest drop-off in the donation journey?
- Which channel converts best into first-time donors?
- Which audiences are most likely to become monthly members?
- What changes would improve conversion and retention?

## Positioning for portfolio use
