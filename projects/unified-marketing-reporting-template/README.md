# Unified Marketing Reporting Template - HopeTrack International

## Positioning

This portfolio project is designed to demonstrate the exact kind of interim reporting solution a **Data & Analytics Specialist** can deploy quickly for a global NGO before a full BI integration is in place.

It is intentionally positioned as more than a spreadsheet:
- a multi-source reporting layer
- a KPI framework across the full funnel
- a decision-support tool for non-technical stakeholders
- a bridge toward future Looker Studio / internal BI implementation

## EN

This project simulates a unified reporting workbook for a fictional NGO, **HopeTrack International**, showing how a data analyst can consolidate multi-source marketing data, standardize interpretation, and turn it into simple, actionable decisions for non-technical stakeholders.

The workbook consolidates 4 data sources:
- social media
- web analytics (GA4-style)
- email campaigns
- donor CRM

It also includes:
- a Start Here onboarding page for users
- a UTM link generator with naming conventions
- a funnel-based KPI dashboard
- an auto-filled weekly report
- a monthly report
- a data quality log
- a channel comparison sheet
- a KPI framework sheet with definitions and business use
- governance notes covering assumptions, limits and recommended actions
- dedicated Looker Studio source tabs for semi-automated dashboard building
