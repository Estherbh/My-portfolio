# Two Strategic Projects To Add Next

These are the two strongest additions if the target role expects fundraising, communications, CRM, and cross-channel measurement.

## 1. Donor Journey & Membership Funnel Analysis

### Goal
Show that you can analyze conversion from acquisition to donor retention, not just web traffic.

### Best format
- Google Sheets workbook
- or Looker Studio dashboard built from a simple prepared dataset

### Data sources to simulate
- GA4 landing page sessions
- campaign UTMs
- email clicks
- CRM donor records
- one-time vs monthly donor outcomes

### Minimum tabs or views
- funnel overview
- first donation conversion
- repeat donor retention
- monthly membership conversion
- channel-to-donor source summary
- data dictionary / assumptions

### Insights to include
- where the biggest drop-off happens
- which channel drives the highest first-donation rate
- which audience segment is most likely to become monthly donors
- what should be optimized next

### Why this project matters
It directly closes the biggest portfolio gap for roles that mention donor journey, CRM, and conversion.

## 2. Cross-Channel Campaign Insights One-Pager

### Goal
Show that you can translate campaign data into simple recommendations for non-technical teams.

### Best format
- one-page HTML report
- or one-page PDF exported from a designed report

### Structure
- campaign objective
- snapshot of channel performance
- 3 key insights
- 3 recommended actions
- short note on measurement limitations

### Channels to compare
- web / GA4
- email
- social
- paid
- donor CRM outcomes if available in the fictional dataset

### Why this project matters
It demonstrates storytelling with data, cross-channel interpretation, and business communication, which are often the deciding factors in communications analytics roles.

## Recommended build order

1. Donor Journey & Membership Funnel Analysis
2. Cross-Channel Campaign Insights One-Pager

If time is tight, build them with fictional HopeTrack data so the portfolio stays consistent and safe to publish.
