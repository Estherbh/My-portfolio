# Cross-Channel Campaign Insights One-Pager

## Overview

This project is a one-page stakeholder report designed to show how campaign data can be translated into clear, business-ready recommendations.

Instead of acting like a dashboard, it acts like a decision memo: what happened, what matters, and what should happen next.

## What this project shows

- cross-channel performance interpretation
- concise reporting for non-technical stakeholders
- practical storytelling with data
- recommendations grounded in campaign evidence

## Files

- Report page: [index.html](index.html)
- Sample dataset: [campaign-summary.csv](campaign-summary.csv)

## Reporting goal

Summarize a quarterly fundraising campaign across:
- web / GA4-style traffic data
- email performance
- paid media contribution
- social engagement
- donor outcomes

## Why this matters in the portfolio

Many analytics roles require more than dashboards. They require someone who can tell a team:
- what changed
- why it matters
- what to do next

That is the purpose of this project.
